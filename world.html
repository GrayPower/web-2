TERRA EXPLORATION

泰拉世界观档案 // ENCYCLOPEDIA OF TERRA

01 // TERRA

泰拉大地


我们所处的这片大地被称为“泰拉”。这里充满了未知与危险，各个种族在这片广袤的土地上繁衍生息，建立了不同的国家与势力。

然而，大地并不平静。天灾的威胁始终笼罩在泰拉之上，迫使人们不断迁徙，寻找安全的栖息之所。

02 // ORIGINIUM

源石


一种黑色的半透明结晶矿物。源石蕴含着巨大的能量，是推动泰拉工业文明发展的核心能源。通过源石技艺（Arts），人们可以施展出类似魔法的奇特能力。

但源石也是诅咒的根源。长期接触源石或被天灾波及，会导致被称为“矿石病”的不治之症。感染者虽然能获得更强的源石技艺，但最终会走向死亡，并成为新的感染源。

03 // CATASTROPHES

天灾


泰拉世界频繁发生的各种自然灾害的统称。风暴、陨石、洪水等灾害中往往伴随着源石的出现。

天灾的破坏力极强，且具有不可预测性。为了在天灾中生存，人类发展出了移动城市技术，以便在灾难来临前进行躲避。

04 // NOMADIC CITIES

移动城市


建立在巨大可移动地基上的城市。通过无数板块的拼接，这些城市可以在荒野上航行，避开天灾的路径。

移动城市是泰拉文明的主要载体，拥有完善的工业、居住和防御系统。不同城邦之间既有贸易往来，也存在着复杂的政治博弈。
